# Ελληνικές Μεταφράσεις - Mountain Car RL Algorithm

Όλα τα αρχεία οδηγιών έχουν μεταφραστεί στα ελληνικά.

## 📄 Μεταφρασμένα Αρχεία

### 1. README_EL.md
- **Περιγραφή**: Βασικός οδηγός για το project
- **Περιεχόμενα**: 
  - Γρήγορη έναρξη
  - Δομή project
  - Λεπτομέρειες περιβάλλοντος
  - Απαιτήσεις

### 2. VISUALIZATION_GUIDE_EL.md
- **Περιγραφή**: Οδηγός για τα διαδραστικά visualization tools
- **Περιεχόμενα**:
  - Πώς να χρησιμοποιήσετε τα dashboards
  - Real-time agent viewer
  - Αντιμετώπιση προβλημάτων
  - Παραδείγματα χρήσης

### 3. REPORT_EL.md
- **Περιγραφή**: Αναλυτική αναφορά project
- **Περιεχόμενα**:
  - Εισαγωγή και περιγραφή περιβάλλοντος
  - Observation, Action, State Spaces
  - Δομή Rewards
  - Επιλογή και σύγκριση αλγορίθμων
  - Εξήγηση κώδικα
  - Αποτελέσματα και ανάλυση
  - Συμπεράσματα

## 🔄 Αρχεία Κώδικα

Τα αρχεία Python (`mountain_car_ppo.py`, `visualize_training.py`, `view_agent.py`) παραμένουν στα αγγλικά καθώς:
- Ο κώδικας πρέπει να είναι στα αγγλικά για συμβατότητα
- Τα σχόλια στον κώδικα μπορούν να είναι στα ελληνικά αν χρειάζεται
- Οι οδηγίες χρήσης είναι στα ελληνικά στα αρχεία οδηγιών

## 📚 Πώς να Χρησιμοποιήσετε

1. **Για γρήγορη έναρξη**: Διαβάστε το `README_EL.md`
2. **Για visualizations**: Δείτε το `VISUALIZATION_GUIDE_EL.md`
3. **Για αναλυτική πληροφόρηση**: Μελετήστε το `REPORT_EL.md`

## 🌐 Διγλωσσία
